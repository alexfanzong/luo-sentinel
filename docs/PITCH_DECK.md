# LUO Sentinel — Pitch Deck Outline

1. **Title** — LUO Sentinel: evidence-bound decision record for RWA on
   Injective Testnet.
2. **Failure mode** — AI agents can compress cross-border legal divergence into
   false certainty.
3. **Core thesis** — the missing layer is not another fluent answer; it is a
   verifiable action boundary before an agent moves toward execution.
4. **User and trigger** — an agent proposes a tokenized treasury distribution.
5. **Product** — Atlas Canvas preserves United States, Hong Kong, Singapore,
   and European Union as distinct signals, or narrows to one jurisdiction when
   the user only asks for that scope.
6. **Verified restraint** — unsupported questions are refused or routed back to
   a reviewed scope instead of receiving a fabricated compliance map.
7. **Agent Review Council** — Scope, Source, and Risk agents produce
   deterministic scorecards over the selected evidence pack before any human
   decision; the score rubric is coverage, authority fit, claim support, and
   residual risk.
8. **Human decision** — a person holds for counsel or prepares a safe,
   non-value-moving testnet decision receipt.
9. **Injective path** — content stays off-chain; the public receipt hash,
   reviewer-controlled wallet action, and runtime-verified contract create a
   replayable responsibility record without putting a private key in the
   product.
10. **Bounded downstream action** — the downstream agent consumes only the
   verified scope, reviewer objections, decision, receipt proof, and unresolved
   constraints, then outputs a scoped counsel-preparation checklist.
11. **Why now** — RWA, stablecoins, custody, and cross-border payments require
    evidence before execution.
12. **Roadmap** — replace deterministic reviewers with live legal/compliance
    agents, add reputation/challenge history, more reviewed scenarios, and
    evidence-quality validation.
